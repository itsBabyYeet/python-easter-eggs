Barry as FLUFL

Joke
An April Fool's joke (PEP 401) that replaces != with the diamond operator <>.
from __future__ import barry_as_FLUFL
print(1 <> 2)
True

Infinite Hash

Math
Python's hash for infinity is the first 6 digits of Pi. A mathematical Easter egg.
hash(float('inf'))
314159
